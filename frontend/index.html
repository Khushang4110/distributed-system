About The Project

This project demonstrates a distributed system built with modern technologies. The solution integrates a responsive frontend, a robust Python backend using Flask, and a MySQL database. All components are containerized with Docker and orchestrated via Kubernetes. Monitoring is implemented using Prometheus and Grafana, while GitHub ensures version control and continuous integration.
